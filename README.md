# wvue

> A Vue.js project

## Build Setup

``` bash
vue搭建项目，里面建立了server，写的是express设置接口，前端通过接口获取数据

# install dependencies
npm install

# 启动express服务
node index

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
